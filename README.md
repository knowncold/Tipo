# Tipo

## Installation

### Requirments

 * Python
   - virtualenv
   - python2.7
   - flask
   - flask-mongoengine
   - requests
   - markdown
 * MongoDB

### Run

```bash
$ virtualenv venv
$ source venv/bin/activate
$ pip install -r requirenments.txt
$ python run.py
```

## Features

 - 支持主页，归档，About三个主页面
 - 支持基于标签，月份，分类的归档列表
 - 支持使用`Markdown`书写并上传博客
 - 支持最近更新，最新评论，月份归档计数，标签云侧边栏功能
 - 翻墙情况下，支持评论功能
